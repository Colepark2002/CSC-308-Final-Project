1. Maximize the window
2. Select a problem from problem drop-down on the menu bar
3. To solve the problem, begin to create a UML diagram on the right panel by clicking and entering a Class name.
4. Left click on class boxes to add methods and variables
5. Add connections between the boxes using the connections button.
6. Test your solution to the problem with the buttons on the bottom right hand side.
7. Check your proficiency with the proficiency button. Problems 2 and 3 can only be completed after the previous
    problems have been correctly submitted.

You can also create and load your own problems by clearing the screen, then creating your desired UML.
    Next, save the problem from the menu bar with a file name and proficiency. Then, select "load" in
    the menu bar and enter the file name to load the code for your problem in the text area.
